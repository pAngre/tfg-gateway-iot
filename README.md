## Descripción

Este repositorio contiene el código fuente del TFG, cuyo objetivo es el diseño
e implementación de un gateway IoT capaz de realizar inferencia de Machine
Learning en el edge utilizando un sistema Linux embebido construido con el
Yocto Project.
